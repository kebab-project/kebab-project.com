Kebab is not a product for end users. Kebab is development platform for developers. If a developer want to develop a cloud, saas, cdn, RESTfull, ajax based, multi-tenant application, kebab is for him/her.

We develops Kebab for us! We wanted a platform which helps us easily, faster,
not hurt when developing web applications.
